Software Arena

Software Arena extends Chatbot Arena with powerful code execution capabilities, enabling evaluation of LLM-generated programs across a wide range of outputs - from simple computations to complex visual interfaces.

What is Software Arena?

Software Arena introduces a plug-and-play code execution environment for Chatbot Arena. It enables direct evaluation of LLM capabilities in:

  • General-purpose code execution across multiple languages
  • Output visualization ranging from text to interactive UIs
  • Collection of human feedback on code quality and execution results

Output Visualization

Software Arena can visualize various types of code execution outputs:

  • Documents (Markdown or Plain Text)
  • Websites (single page HTML)
  • Scalable Vector Graphics (SVG) images
  • Plots
  • Tables
  • Interactive React/Vue components
  • PyGame
  • Gradio/Streamlit applications
  • Mermaid diagrams

Technical Implementation

Software Arena builds upon FastChat, the foundation of Chatbot Arena, providing seamless code execution capabilities. The implementation focuses on:

  • Core Code Execution: Secure, sandboxed environment using E2B for executing code in any supported language, with potential Modal integration for improved scalability
  • Output Handling: Universal output capture and rendering system for all types of program output
  • Execution Tracking: Comprehensive logging of code execution, results, and user feedback

Supported Frameworks

Currently supported software frameworks include:

  • Python Code Interpreter
  • React (https://react.dev/)
  • Vue.js (https://vuejs.org/)
  • Gradio (https://www.gradio.app/)
  • NiceGUI (https://nicegui.io/)
  • Pygame via Pygbag (https://pygame-web.github.io/wiki/pygbag/)
  • Streamlit (https://streamlit.io/)
